🔐 Projet : Gestion des droits Active Directory via interface Web

🎯 Objectif

Créer une interface web sécurisée permettant de rechercher des utilisateurs et des groupes dans l’Active Directory du domaine ATE (et autres), avec exportation automatique des résultats en CSV et affichage structuré des informations.

⚙️ Technologies utilisées

  • PowerShell — Scripts pour interagir avec Active Directory
  • PHP — Exécution des scripts et redirection
  • HTML/CSS — Interface web (formulaires, tableaux dynamiques)
  • JavaScript — Chargement et affichage dynamique des résultats CSV
  • WAMP — Serveur local (Apache, PHP, MySQL)

📂 Fonctionnalités principales

  • 🔍 Recherche d’un utilisateur dans une OU spécifique ou dans tout le domaine
  • 🔐 Affichage des groupes associés à un utilisateur
  • 📁 Recherche d’un groupe et affichage des membres
  • 📤 Export automatique des résultats en fichiers CSV
  • 📊 Affichage des détails de l’utilisateur : statut, dernière connexion, date de modification du mot de passe
  • 📜 Interface claire avec filtrage en temps réel

📌 Exemple de cas d’usage

Un administrateur souhaite vérifier les groupes d’un utilisateur dans l’OU DDI45. Il entre son nom dans le formulaire web, ce qui déclenche un script PowerShell. Le résultat s’affiche en tableau avec tous les groupes liés et un bouton permet d’accéder aux détails (compte actif, dernières connexions...).
