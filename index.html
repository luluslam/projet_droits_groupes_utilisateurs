Projet Active Directory — Luna MEHDI

Exploration et gestion des utilisateurs, groupes et OU via un portail web connecté à PowerShell.

Fonctionnalités Structure Scripts

📋 Présentation du projet

Ce projet a été développé dans le cadre de ma formation BTS SIO SLAM.
Il permet l’interaction en temps réel avec l’Active Directory à travers une interface web :

  • 🔎 Recherche d’utilisateurs dans tout le domaine ou dans une OU précise
  • 🧑‍🤝‍🧑 Affichage des groupes et des membres
  • 📂 Visualisation dynamique des sous-OU
  • 📊 Export CSV automatique des résultats
  • ⚙️ Détails d’un compte : activité, dernière connexion, changement de mot de passe

🧱 Architecture

  • Frontend : HTML / CSS
  • Backend : PHP + PowerShell
  • Serveur : WAMP (hébergé sur VM)
  • Domaine : `ate.at.int` / `pref45.int`

📦 Exemple de technologies utilisées

PowerShell Active Directory WAMP CSV HTML/CSS PHP Windows Server OU / Groupes
